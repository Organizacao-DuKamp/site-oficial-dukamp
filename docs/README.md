# Documentação do Projeto Dukamp

Este diretório reúne a documentação técnica em português brasileiro do e‑commerce **Dukamp**, construído sobre a stack **TanStack Start + React 19 + Tailwind v4 + Lovable Cloud (Supabase)**.

## Índice

| Arquivo | Conteúdo |
| --- | --- |
| [architecture.md](./architecture.md) | Visão de arquitetura, camadas e runtime |
| [project-structure.md](./project-structure.md) | Explicação de cada pasta e arquivos importantes |
| [setup.md](./setup.md) | Como instalar, configurar e rodar o projeto |
| [features.md](./features.md) | Funcionalidades principais e regras de negócio |
| [data-flow.md](./data-flow.md) | Fluxo de dados entre UI, server functions e banco |
| [components.md](./components.md) | Componentes reutilizáveis e estrutura de UI |
| [api-and-integrations.md](./api-and-integrations.md) | APIs, integrações externas e variáveis de ambiente |
| [maintenance.md](./maintenance.md) | Como estender/modificar com segurança |

## Como ler

1. Comece por `architecture.md` para entender o modelo geral.
2. Use `project-structure.md` como mapa quando precisar localizar arquivos.
3. Consulte `data-flow.md` antes de mexer em qualquer feature que envolva banco de dados ou autenticação.
